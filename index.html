연결 상태

Apps Script Web App URL은 페이지 설정 파일에서 1회 관리합니다. 사용자는 URL과 Token을 입력하지 않고 발주서 파일만 선택해 실행합니다.

Apps Script에는 ALLOW_PAGE_UPLOAD=true 스크립트 속성이 필요합니다. API Token은 GitHub Actions 호출용으로만 유지합니다.

실제 처리 흐름

업로드 버튼 클릭 후 Apps Script가 처리하는 순서입니다.

  1. 1. 발주서 파일 첨부xlsx / xls / csv
  2. 2. 파일 형식 검사확장자 확인
  3. 3. 브랜아크 Apps Script 호출자동 연결
  4. 4. Google Drive 저장발주서 폴더
  5. 5. 단가표 접근 확인PRICE_SHEET_ID
  6. 6. 일일 마감 양식 작성DAILY_SHEET_ID
  7. 7. 작업 결과 화면 표시성공 / 오류

발주서 업로드 및 일일마감 작성

발주서 파일을 선택하고 주문 정보를 확인한 뒤 실행합니다. URL과 Token은 화면에서 입력하지 않습니다.
