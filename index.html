BRANARK CLOSING LEDGER AUTOMATION

브랜아크 발주서 파일 분석 및 일일마감 자동 반영

발주서 파일을 업로드하면 브랜아크 전용 Apps Script가 파일을 저장하고, 시트를 읽고, 단가표를 대조한 뒤 일일마감 반영 예정 행까지 정리합니다.

안내

현재 운영 기본값은 자동 분석 모드입니다. 상품명, 수량, 공급단가를 화면에서 따로 입력하지 않고 파일 내용으로만 처리합니다.

금액은 공급단가표를 기준으로 검증하며, 단가표에서 찾지 못한 상품이나 수량 불일치가 있으면 일일마감 반영을 차단합니다.

2

발주서 업로드

지원 형식은 csv, xlsx, xls이며 8MB 이하 파일만 업로드할 수 있습니다.

처리 흐름

발주서 파일 업로드 → 파일 저장 및 시트 탐색 → 상품별 집계 → 공급단가표 검증 → 일일마감 반영 예정 행 구성 → 검증 통과 시 일일마감 시트 반영

운송장이 있으면 운송장을 기준 수량으로 사용하고, 출고일지/출고일지(2)는 검증용 비교 데이터로만 사용합니다.

  1. 파일 조건 확인대기
  2. Apps Script 요청 전송대기
  3. 파일 분석 및 시트 탐색대기
  4. 수량 비교 및 단가 검증대기
  5. 일일마감 반영 예정 행 구성대기
  6. 일일마감 시트 반영대기

자동화 연결 상태가 정상일 때 업로드 버튼이 활성화됩니다.
