# 브랜아크 일일마감 자동화 화면 명세서

## 화면 목적

발주서 파일을 업로드하고, Google Drive 저장/발주서 검증/일일 마감 양식 반영 상태를 한 화면에서 확인한다.

---

## 1. 메인 화면

```text
┌────────────────────────────────────────────┐
│ 브랜아크 일일마감 자동화                    │
│ 발주서를 업로드하면 일일 마감 양식에 반영됩니다. │
│                                            │
│ [파일 선택]  선택된 파일 없음               │
│                                            │
│ 저장 방식                                  │
│ ● Google Drive에 저장 후 시트 연동          │
│ ○ 파일만 즉시 검토                         │
│                                            │
│ [업로드 및 검증 시작]                       │
└────────────────────────────────────────────┘
```

### 주요 요소

| 요소 | 설명 |
|---|---|
| 파일 선택 | xlsx, xls, csv 파일 업로드 |
| 저장 방식 | 기본값은 Google Drive 저장 후 시트 연동 |
| 업로드 버튼 | 파일 검증 시작 |
| 안내문 | 일일 마감 양식 변경 금지 안내 |

---

## 2. 업로드 진행 화면

```text
┌────────────────────────────────────────────┐
│ 처리 상태                                  │
│                                            │
│ ✅ 파일 형식 확인                           │
│ ✅ 중복 파일명 확인                         │
│ ✅ Google Drive 저장                        │
│ ⏳ 발주서 내용 확인                         │
│ ⏳ 단가표 매칭                              │
│ ⏳ 일일 마감 양식 반영 준비                 │
└────────────────────────────────────────────┘
```

### 상태값

| 상태 | 의미 |
|---|---|
| 대기 | 아직 처리 전 |
| 진행중 | 현재 처리 중 |
| 성공 | 정상 완료 |
| 확인 필요 | 검수 필요 |
| 실패 | 처리 차단 |

## 4. 일일 마감 미리보기

```text
┌────────────────────────────────────────────┐
│ 일일 마감 추가 예정 내용                    │
├────────────┬───────────────┬────┬──────┬──────┤
│ 주문일      │ 상품명          │ 수량 │ 단가 │ 합계 │
├────────────┼───────────────┼────┼──────┼──────┤
│ 2026-07-02 │ 궁채장아찌 4kg  │ 37 │ 8,800 │ ... │
└────────────┴───────────────┴────┴──────┴──────┘
```

### 컬럼

| 컬럼 | 설명 |
|---|---|
| 주문일 | 주문일 또는 참고일 |
| 상품명 | 정규화 상품명 |
| 수량 | 실제 수량 |
| 공급단가 | 단가표 기준 |
| 공급가 합계 | 수량 × 공급단가 |
| 비고 | 검증 상태 |

## 6. 오류 화면

```text
┌────────────────────────────────────────────┐
│ 처리할 수 없습니다                          │
│                                            │
│ ❌ 중복된 파일명입니다.                     │
│                                            │
│ 같은 이름의 파일이 이미 Google Drive에 있습니다.│
│ 파일명을 변경하거나 기존 파일을 확인해주세요.   │
│                                            │
│ [다시 업로드]                              │
└────────────────────────────────────────────┘
```

## 오류 메시지 목록

| 상황 | 메시지 |
|---|---|
| 중복 파일명 | 중복된 파일명입니다. |
| 형식 오류 | 지원하지 않는 파일 형식입니다. |
| 운송장 없음 | 운송장 시트를 찾을 수 없습니다. |
| 출고일지 없음 | 출고일지 시트를 찾을 수 없습니다. |
| 상품명1 없음 | 상품명1 컬럼을 찾을 수 없습니다. |
| 단가 누락 | 단가표에 없는 상품이 있습니다. |
| 양식 변경 | 일일 마감 양식이 변경되었습니다. |

---

## 7. 모바일 화면 규칙

- 카드형 레이아웃으로 세로 배치
- 표는 가로 스크롤 허용
- 주요 버튼은 화면 하단에서 크게 노출
- 오류/확인 필요 메시지는 상단 고정 노출

---

## 8. 화면 우선순위

1. 파일 업로드
2. 검증 결과
3. 일일 마감 미리보기
4. 단가표 검증
5. 자동화 규칙 안내
