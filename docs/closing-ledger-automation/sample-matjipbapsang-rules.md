# 브랜아크_맛집밥상 발주서 기준 마감원장 처리 규칙

업로드 예시 파일:

- `브랜아크_맛집밥상_발주서_260702_발주용.xlsx`
- `공급단가.XLS`
- `일간 마감정산 양식.xlsx`

이 문서는 위 파일 구조를 기준으로 마감원장 자동화에서 반드시 반영해야 할 처리 기준을 정리합니다.

---

## 1. 기준 파일과 역할

| 파일 | 역할 |
|---|---|
| `브랜아크_맛집밥상_발주서_260702_발주용.xlsx` | 발주서 원본. 운송장/출고일지 기준 검증 대상 |
| `공급단가.XLS` | 품목별 공급단가 기준 파일 |
| `일간 마감정산 양식.xlsx` | 최종 마감정산 출력 양식 |

---

## 2. 발주서 기준 시트

### 2-1. 운송장 시트

- `운송장` 시트의 G열 `상품명1`을 기준으로 품목별 수량을 집계합니다.
- 옵션명 뒤 `_숫자`는 실제 개수로 계산합니다.
  - 예: `아삭아삭 궁채장아찌 1kg_10` → `아삭아삭 궁채장아찌 1kg` 10개
  - 예: `김밥집 궁채 2kg_4` → `김밥집 궁채 2kg` 4개
- `_숫자`가 없으면 기본 수량은 1개로 계산합니다.
- `수량(A타입)` 열은 기본적으로 1로 유지되는 경우가 많으므로, 실제 발주 수량은 `상품명1` 옵션값을 우선 반영합니다.

### 2-2. 출고일지 시트

- `출고일지` 시트에는 운송장 기준으로 집계된 품목별 수량이 정리되어 있습니다.
- 자동화 결과는 `운송장` 집계값과 `출고일지` 집계값이 일치하는지 반드시 검증해야 합니다.
- 일부 파일은 `출고일지`와 `출고일지(2)`로 나뉘어 있을 수 있습니다.
  - 이 경우 두 시트를 모두 읽어 합산합니다.
  - 합산 결과가 `운송장` G열 기준 집계와 일치해야 합니다.

---

## 3. 일치 여부 확인 규칙

자동화는 아래 2개 결과를 비교합니다.

1. `운송장` 시트 G열 `상품명1` 기준 품목별 실제 수량
2. `출고일지` / `출고일지(2)` 시트의 품목별 수량

### 검증 결과 상태

| 상태 | 의미 |
|---|---|
| `일치` | 운송장 집계 수량과 출고일지 수량이 동일 |
| `불일치` | 품목별 수량 차이 발생 |
| `출고일지 누락` | 운송장에는 있으나 출고일지에 없는 품목 |
| `운송장 누락` | 출고일지에는 있으나 운송장에 없는 품목 |

불일치가 있는 경우 PR 검증 리포트에 반드시 노출합니다.

---

## 4. 날짜 기준

### 확인해야 하는 날짜

`출고일지` 시트에서 아래 날짜를 확인합니다.

- 출고일
- 주문일시 또는 주문일
- 출고요청일

### 마감 기준일

- 마감원장은 `출고일` 기준으로 정리합니다.
- `출고요청일`과 `출고일`이 다를 경우, `출고일`을 우선합니다.
- 주문일은 참고값으로 보관하되, 마감 집계 기준일로 사용하지 않습니다.

예시:

| 항목 | 값 |
|---|---|
| 출고일 | 2026-07-02 |
| 주문일 | 2026-07-02 |
| 출고요청일 | 2026-07-02 |
| 마감 기준일 | 2026-07-02 |

---

## 5. 공급단가 매칭 및 금액 계산

`공급단가.XLS` 파일을 기준으로 품목별 공급단가를 매칭합니다.

마감정산에는 아래 계산값이 노출되어야 합니다.

```text
품목별 수량 × 공급단가 = 공급가 합계
```

### 필수 출력 컬럼

| 컬럼 | 설명 |
|---|---|
| 주문일 | 참고용 주문일 |
| 출고일 | 마감 기준일 |
| 상품명 | 정규화된 상품명 |
| 수량 | 출고일 기준 실제 수량 |
| 공급단가 | `공급단가.XLS` 기준 단가 |
| 공급가 합계 | 수량 × 공급단가 |
| 검증상태 | 운송장/출고일지 일치 여부 |
| 비고 | 단가 누락, 수량 불일치 등 |

---

## 6. 일간 마감정산 양식 반영

최종 출력은 `일간 마감정산 양식.xlsx` 구조를 따릅니다.

예시 양식 컬럼:

| 주문일 | 상품명 | 수량 | 공급단가 | 공급가 합계 | 비고 |
|---|---|---:|---:|---:|---|

자동화 생성 결과에는 최소 아래 내용이 들어가야 합니다.

- 출고일 기준 상품별 수량
- 공급단가
- 공급가 합계
- 택배발송 등 배송비 항목이 필요한 경우 별도 행으로 표시
- 단가 누락 또는 검증 불일치 항목은 비고에 표시

## 8. PR에 반드시 포함할 검증 내용

자동화 PR 본문 또는 검증 리포트에는 아래 내용이 상세히 작성되어야 합니다.

### 필수 검증 항목

- 원본 파일명
- 처리 기준 출고일
- 주문일 / 출고요청일 / 출고일 비교
- 운송장 G열 기준 집계 결과
- 출고일지/출고일지(2) 기준 집계 결과
- 두 집계 결과의 일치 여부
- 공급단가 매칭 결과
- 단가 누락 품목
- 공급가 합계 계산 결과
- 일간 마감정산 양식 반영 결과

### PR 본문 예시

```markdown
## 작업 개요

브랜아크_맛집밥상 발주서를 기준으로 마감원장 초안을 생성했습니다.

## 기준 파일

- 발주서: 브랜아크_맛집밥상_발주서_260702_발주용.xlsx
- 공급단가: 공급단가.XLS
- 출력 양식: 일간 마감정산 양식.xlsx

## 날짜 기준

- 주문일: 2026-07-02
- 출고요청일: 2026-07-02
- 출고일: 2026-07-02
- 마감 기준일: 출고일 2026-07-02

## 검증 결과

- 운송장 G열 기준 품목 수량 집계 완료
- 출고일지 기준 품목 수량 확인 완료
- 운송장/출고일지 수량 일치 여부 확인 완료
- 공급단가 매칭 완료
- 공급가 합계 계산 완료

## 확인 필요

- 단가 누락 품목: 0건
- 수량 불일치 품목: 0건
```

## 9. 자동화 실패 또는 확인 필요 조건

아래 조건이 발생하면 Google Spreadsheet를 생성하지 않고 PR에서 확인 필요로 남깁니다.

- 운송장 G열 기준 집계와 출고일지 수량이 불일치
- `출고일지(2)`가 존재하지만 합산되지 않음
- 출고일을 찾을 수 없음
- 공급단가가 누락됨
- 공급가 합계 계산이 불가능함
- 일간 마감정산 양식 컬럼 구조가 변경됨
