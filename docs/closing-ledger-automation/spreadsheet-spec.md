# Google Spreadsheet 마감원장 규격 초안

## 생성 조건

마감원장 Google Spreadsheet는 PR이 main 브랜치에 merge된 이후에만 생성합니다.

```text
PR 생성 → 사람 검증 → merge → Google Spreadsheet 생성
```

---

## 시트 구성

| 시트명 | 설명 |
|---|---|
| `전체 마감원장` | 전체 업체/상품 통합 내역 |
| `업체별 요약` | 거래처별 수량·금액 합계 |
| `단가 누락` | 단가표에 없는 상품 |
| `검증 로그` | 처리 일시, 원본 파일명, 검수자 기록 |
| `원본 데이터` | 발주서 원본 파싱 결과 |

---

## 전체 마감원장 컬럼

| 컬럼 | 설명 | 예시 |
|---|---|---|
| 처리일자 | 마감원장 생성일 | 2026-07-02 |
| 발주일자 | 발주서 기준일 | 2026-07-02 |
| 업체명 | 발주 업체 | 식봄 |
| 원본상품명 | 발주서 원본 상품명 | 아삭아삭 궁채장아찌 1kg_10 |
| 상품명 | 정규화 상품명 | 아삭아삭 궁채장아찌 1kg |
| 규격 | 상품 규격 | 1kg |
| 수량 | 옵션 반영 실제 수량 | 10 |
| 단가 | 발주단가표 기준 단가 | 4,800 |
| 금액 | 수량 × 단가 | 48,000 |
| 매칭상태 | 단가 매칭 상태 | 정상 / 확인 필요 / 누락 |
| 비고 | 검수 참고사항 | 상품명 유사 매칭 |

---

## 업체별 요약 컬럼

| 컬럼 | 설명 |
|---|---|
| 업체명 | 거래처명 |
| 상품수 | 정규화된 상품 종류 수 |
| 총수량 | 전체 발주 수량 |
| 총금액 | 업체별 발주 금액 합계 |
| 단가누락 | 단가 누락 상품 수 |
| 확인필요 | 유사 매칭 또는 예외 상품 수 |

---

## 단가 누락 시트 컬럼

| 컬럼 | 설명 |
|---|---|
| 원본상품명 | 발주서 원본 상품명 |
| 정규화상품명 | 정규화 결과 |
| 수량 | 실제 계산 수량 |
| 누락사유 | 단가표 없음 / 상품명 불일치 / 세트 분리 실패 |
| 조치상태 | 미처리 / 단가 추가 완료 / 제외 처리 |

---

## 서식 규칙

- 단가 누락 행은 빨간색 배경 처리
- 확인 필요 행은 노란색 배경 처리
- 정상 매칭 행은 별도 강조 없음
- 금액 컬럼은 원화 서식 적용
- 총합계 행은 굵게 표시

## 텔레그램 완료 알림 예시

```text
✅ 마감원장 생성 완료

- 처리일자: 2026-07-02
- 발주서: 20260702_식봄_발주서.xlsx
- 총 상품수: 24개
- 총 수량: 1,284개
- 총 금액: 4,523,000원
- 단가 누락: 0건

Google Spreadsheet:
{생성된 시트 링크}
```

```text
⚠️ 마감원장 생성 확인 필요

- 상품명 매칭 확인 필요: 2건

PR에서 검증 후 수정해 주세요.
```
