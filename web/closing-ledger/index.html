BRANARK CLOSING LEDGER AUTOMATION

브랜아크 발주서 파일 분석 및 일일마감 자동 반영

발주서 파일을 업로드하면 브랜아크 전용 Apps Script가 파일을 분석하고, 상품별 집계와 단가 매칭을 거쳐 일일마감 양식에 자동 반영합니다.

운영 기본값

이 페이지는 자동 분석 모드가 기본입니다. 수동 테스트 입력 없이 파일 업로드만으로 분석, 검토, 반영 결과를 확인하도록 구성했습니다.

발주서 업로드

지원 형식은 csv, xlsx, xls이며 업로드 후 자동 분석과 일일마감 반영이 한 번에 진행됩니다.

처리 흐름

발주서 업로드 → 파일 분석 → 상품별 집계 → 단가 매칭 → 일일마감 반영 예정 행 구성 → 검증 통과 시 일일마감 반영

상품명, 수량, 단가를 수동 입력하지 않아도 되며, 파일 내부 데이터 기준으로 자동 정리됩니다.

자동화 연결이 정상이면 업로드 버튼이 활성화됩니다.
