BRANARK CLOSING LEDGER

브랜아크 일일마감 자동화

발주서 파일을 업로드하면 Google Drive에 저장하고, 검증 후 일일 마감 양식에 추가하는 화면 초안입니다.

필수 운영 규칙

일일 마감 양식을 마음대로 변경하지 마세요. 컬럼이 변경되면 자동화가 중단됩니다.

1

발주서 업로드

xlsx, xls, csv 파일만 지원합니다.

현재 화면은 HTML 프로토타입입니다. 실제 Drive 저장/API 연결은 후속 단계에서 진행합니다.

2

처리 상태

파일 업로드 후 단계별 상태를 표시합니다.

  1. 파일 형식 확인대기
  2. 중복 파일명 확인대기
  3. Google Drive 저장대기
  4. 발주서 내용 확인대기
  5. 단가표 매칭대기
  6. 일일 마감 양식 반영 준비대기
3

오류 메시지 규칙

중복된 파일명입니다. 지원하지 않는 파일 형식입니다. 운송장 시트를 찾을 수 없습니다. 출고일지 시트를 찾을 수 없습니다. 상품명1 컬럼을 찾을 수 없습니다. 일일 마감 양식이 변경되었습니다.
